# Chat Agent

Agentic Flow project generated by [AgenticTranscoder](https://github.com/daiichisankyo/AgenticFlow).

## Quick Start

```bash
# Install dependencies
uv sync

# Start backend
uv run uvicorn server:app --port 8000

# Start frontend (new terminal)
cd frontend
npm install
npm run dev
```

- Backend: http://localhost:8000
- Frontend: http://localhost:5173

## Architecture

This project uses **Agentic Flow** patterns:

- `phase()` for conversation flow
- `Runner` for flow execution
- `ChatKitServer` for frontend integration
- `SQLiteStore` for thread persistence
